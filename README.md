# Grimstone

Official website for Grimstone - a punk/pop-punk/alternative rock band.

## About

Grimstone plays energetic live shows featuring punk, pop-punk, and 90s-2000s alternative rock. Influenced by Jimmy Eat World, Pixies, Ramones, Foo Fighters, The Strokes, The Killers, and Franz Ferdinand.

## Website

Visit us at: [grimstonemusic.com](https://grimstonemusic.com)

## Tech Stack

- Pure HTML/CSS/JavaScript
- Custom audio player with Web Audio API visualizer
- Hosted on GitHub Pages
- Responsive design for mobile and desktop

## Local Development

```bash
# Serve locally with Python
python3 -m http.server 8000

# Or with PHP
php -S localhost:8000
```

Visit http://localhost:8000

## Deployment

Push to `main` branch to deploy automatically via GitHub Pages.

```bash
git add .
git commit -m "Update site"
git push origin main
```
